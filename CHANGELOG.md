# Changelog

## v1.0.0 — 2026-05-13

Initial release.

### Features
- Connect to PostgreSQL 13+ and SQL Server 2017/2019/2022/Express
- Browse all tables with columns, data types, nullability, defaults, and primary key markers
- View all indexes with column coverage and uniqueness flags
- View all foreign key relationships
- View full SQL definition for every database view
- Instant search/filter across tables and views
- PDF export via browser print
- License key activation (one-time purchase, self-hosted)
- Self-hosted — credentials and schema never leave your machine
